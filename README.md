## 安装

```bash
npm i sunshj -g

sun --version
```

## 创建预设github模板

```bash
sun create my-app
```

## 克隆项目

```bash
sun get sunshj/vue3-ts-starter

#or
sun get sunshj/vue3-ts-starter#main my-app
```

## 项目配置

```bash
sun config

# or
sun config --eslint --prettier --stylelint --lintStaged --commitlint -w
```

## 创建Node.js项目(本地模版)

```bash
sun new my-app
```
